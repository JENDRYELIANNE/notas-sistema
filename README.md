⚪Sistema de Cálculo de Notas de Estudiantes
------------------------------------------------------------
Descripción

Este proyecto trata de una aplicación de escritorio desarrollada en C#, que permite capturar las cuatro notas de un estudiante, calcular su promedio, y determinar si necesita completivo o extraordinario, siguiendo las reglas académicas establecidas.
El sistema evalúa automáticamente el resultado final del estudiante y muestra si aprueba o reprueba, activando dinámicamente los campos necesarios según el caso.

⚪Tecnologías Utilizadas
------------------------------------------------------------
•Lenguaje: C#

•Framework: .NET (Windows Forms)

•Entorno de Desarrollo (IDE): Visual Studio

•Base de Datos: No aplica

⚪Funcionalidades del Sistema
------------------------------------------------------------
•Captura de 4 notas del estudiante
•Cálculo automático del promedio
•Activación automática del campo Completivo si aplica
•Activación automática del campo Extraordinario si aplica
•Validaciones de notas
•Resultado final claro (Aprobado / Reprobador)

⚪Uso o Ejecución
------------------------------------------------------------
1-Abrir el proyecto en Visual Studio
2-Ejecutar el formulario principal
3-Ingresar las 4 notas del estudiante
4-Presionar el botón Calcular
5-El sistema indicará si necesita completivo o extraordinario y mostrará el resultado final
